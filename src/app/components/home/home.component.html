OPERATOR DESCRIPTION
map Observable operator that allows you to transform data in the Observable chain
tap allows us to perform some action without changing the response. It does not stop, block or change Observable chain. It just runs some code with the data (response) you get back from the Observable -- allows us to execute some code here in place without altering the data that is thuneled throug the Observable
take Emits only the first count values (selected number) emitted by the source Observable
filter Filter items emitted by the source Observable by only emitting those that satisfy a specified predicate.
